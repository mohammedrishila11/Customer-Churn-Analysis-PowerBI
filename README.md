# 📊 Customer Churn Analysis Dashboard using Power BI

## 📌 Project Overview

Customer churn is a major challenge for telecom companies as retaining existing customers is more cost-effective than acquiring new ones.

This project presents an interactive Power BI dashboard that analyzes customer behavior, identifies the key factors contributing to customer churn, and provides actionable insights to support better business decisions.

The dashboard is designed to help stakeholders understand customer retention patterns and improve customer loyalty through data-driven strategies.

---

## 🎯 Business Objective

The objectives of this project are to:

- Analyze customer churn behavior
- Identify high-risk customer segments
- Understand factors affecting customer retention
- Monitor revenue and customer performance
- Support business decision-making using interactive dashboards

# 📊 Key Insights

✔ Customers with Month-to-Month contracts have the highest churn rate.

✔ Customers within their first year are more likely to leave.

✔ Fiber Optic customers experience higher churn compared to DSL users.

✔ Customers without Online Security and Tech Support show increased churn.

✔ Higher monthly charges are associated with higher customer churn.

---

# 💼 Business Recommendations

- Promote long-term contracts.
- Improve customer onboarding.
- Offer loyalty programs for new customers.
- Bundle Online Security and Tech Support services.
- Review pricing for high-charge customers.
